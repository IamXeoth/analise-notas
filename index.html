SISTEMA DE NOTAS

ECIT HENRIQUE FERNANDES DE FARIAS

Sistema automatizado para análise de desempenho acadêmico. Faça upload de uma planilha Excel/CSV ou adicione dados manualmente. O sistema calculará automaticamente todas as estatísticas necessárias para relatórios educacionais.

Como usar:

1

Planilha Excel/CSV

Formato: Coluna 1: ID, Coluna 2: Nome, Colunas 3+: Notas, Última coluna: Frequência (Frequente/Infrequente - opcional)

2

Entrada Manual

Adicione estudantes individualmente com ID, nome, nota e status de frequência

3

Análise Automática

Sistema gera relatórios completos: aprovação, frequência, médias e indicadores SEE-PB

📊 Planilhas: Use formato Excel ou CSV com colunas organizadas. 📝 Entrada Manual: Controle total sobre cada dado inserido. 📈 Relatórios: Análise completa automatizada.
